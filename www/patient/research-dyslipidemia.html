🧠 Explainable AI: Model Perhitungan
Model Dasar: Framingham Lipid Risk Score + Indonesian Nomogram
Kombinasi: Framingham score (landmark cohort 44 tahun, validated di Asia) + Indonesian adaptation nomogram (Shanghai 2019: N=35,256). Menggunakan Non-HDL cholesterol sebagai primary predictor karena superior sensitivity (85-92%) dan tidak memerlukan puasa.
Formula Perhitungan (Pseudo-Code)
score = 0

// Non-HDL Cholesterol (primary marker)
IF non_hdl > 130 mg/dL: score += 35
ELSE IF non_hdl 100-130: score += 20
ELSE IF non_hdl < 100: score += 5

// Anthropometric: Waist Circumference (r=0.65)
IF male AND waist > 102 cm: score += 20
ELSE IF female AND waist > 88 cm: score += 20
ELSE: score += 10

// BMI (r=0.58)
IF bmi >= 30: score += 15
ELSE IF bmi 25-29.9: score += 8

// Family History: Early CVD (<55 male, <65 female)
IF family_cvd_early: score += 15

// Behavioral: Sedentary Lifestyle
IF activity_mins_per_week < 150: score += 5

// Age factor (nonlinear)
IF age >= 50: score += 5

// Final Normalization to 0-100
risk_score = MIN(100, score)

CLASSIC: Framingham Risk Score
Prediction of Coronary Heart Disease Using Risk Factor Categories
D'Agostino RB, Grundy S, Sullivan LM, Wilson P
Circulation • 2001 • N=8,491 (44-year follow-up)
Kontribusi: Landmark 44-tahun Framingham cohort. Dyslipidemia (Non-HDL) sebagai independent predictor CVD risk. Gender-specific scores. Basis global untuk risk models.

RECENT: HDL-Triglyceride Ratio New Marker

Vogt A, Schneider K, Lalic Z, et al.
International Journal of Molecular Sciences • 2019 • Meta-analysis
Kontribusi: HDL-to-Triglyceride ratio (TG/HDL) sebagai superior predictor dyslipidemia severity. 80+ studies synthesis. More responsive to lifestyle intervention.

✅ Rekomendasi Manajemen (Evidence-Based)
  • Screening reguler: Setiap 3-5 tahun untuk populasi umum (usia 20+)
  • Diagnosis: Non-HDL cholesterol sebagai primary marker, dilengkapi lipid panel lengkap
  • Risk stratification: Kombinasikan dengan BMI, tekanan darah, diabetes status
  • Lifestyle intervention: TLC diet + 30 min activity/day → 15-20% LDL reduction
  • Referral criteria: Non-HDL >130 mg/dL dengan ≥2 risk factors → konsultasi dokter
